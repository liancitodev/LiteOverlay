## Stream Widget Lite - TikTok & Twitch Overlay

A high-performance desktop overlay tool designed for single-monitor streamers, optimized for real-time TikTok Live interaction and Minecraft server integration.

### Key Features

* **Ghost Mode:** Allows the window to be locked, making it "invisible" to mouse clicks. This enables players to view their chat or alerts on top of the game in full-screen mode without interfering with gameplay controls.
* **Intelligent TTS Reader:** Advanced Text-to-Speech system for chat messages featuring automatic filters, bot-ignoring support, and cleanup of links or unnecessary symbols.
* **Professional Web Integration:** Utilizes high-performance WebView2 technology to render custom panels, including like counters, top donors, 3D cube systems, and live TikTok wheels.
* **Minecraft Synchronization:** Built-in WebSocket support to capture in-game events in real-time, allowing direct interaction between TikTok events and Minecraft actions.
* **Discord Notifications:** Automated linking via Webhooks to send logs of winners or major events directly to dedicated Discord channels.
* **Resource Optimization:** Engineered for minimal CPU and RAM usage, ensuring no frame rate drops during intensive gaming sessions.

### Instructions for Use

1. Download the **LiteOverlay.exe** file from the Assets section.
2. Run the application (no installation required).
3. Access the settings menu and paste your custom Widget URL provided by the bot.
4. Configure the TTS volume and window opacity to your preference.
5. Enable the window lock to activate Ghost Mode before launching your game.
